Growth story

My journey with programming is not yet a long one, but small steps towards professionalism are already there. Below little milestones to showcase the cultivation of new skills during my studies.

The first GUI

Python was the first programming language to be tackled. It taught me the very basics of programming. As a result I could create my first clumsy GUI.

The second GUI

After learning the basics of Python it was time to explore C++. The second programming course led me from value semantics to dynamic memory management. As a result I created my second GUI: a simple memory game.

A peep into Embedded Systems

The introductory course on embedded systems gave me a chance to familiarize myself with C, Rust and Assembly languages. During the course we created a simple alien shooter game for a led matrix in student pairs.

Smarter code, better performance

A course on Datastructures and Algorithms taught me the principles of choosing better data structures, keeping performance in mind. The course assignment challenged to create an entire class in C++ with various algorithm implementations.
